## Seyahat Kayıt Sistemi

ReactJS, Vite ve Tailwind CSS kullanılarak geliştirilmiş modern bir seyahat kayıt uygulaması.

## Canlı Demo 
https://seyahat-kaydi.netlify.app

## SS'ler Ekran Görüntüsü Dosyasında 

## Özellikler
-  Seyahat kaydı ekleme
-  Kayıtları listeleme
-  Kayıt güncelleme
-  Kayıt silme
-  Koyu /  Açık tema desteği
-  Kayıtlarda arama
-  İstatistik kartları (toplam kayıt, ülke, şehir)

## Kullanılan Teknolojiler
- ReactJS
- Vite
- Tailwind CSS
